# Vision

## Core Idea
Create a powerful personal assistant framework by combining Obsidian as the human-facing knowledge environment with AI as an operational layer.

## Desired Outcome
- A reusable Obsidian vault scaffold for AI-assisted work across private life, work, projects, hobbies, finance, and other domains.
- A framework that remains usable across different AI tools because context lives in files instead of sessions.
- A public, free, low-cost foundation others can adopt and extend.

## Product Principles
- Obsidian as the primary interface for knowledge and structure
- AI as collaborator, operator, and assistant
- No dependency on one single vendor or client
- Strong preference for transparent files over hidden context
- Safety-aware but honest about limits and responsibility

## V1 Focus
- Governance
- File structure
- Reusable templates
- Onboarding and configuration
- Generic skills
