# Vault Maintenance Skill

## Activation
- Consistency checks
- Link maintenance
- Small structural cleanup
- Governance and template maintenance

## Working Style
1. Read relevant files and links before changing them.
2. Prefer small, targeted corrections.
3. Avoid unrequested large-scale rewrites.
4. Check affected dependencies after changes.
5. Call out open issues explicitly.

## Focus
- Missing or outdated references
- Inconsistent naming patterns
- Missing standard files in structured topics
- Gaps that prevent AI-agnostic continuation
