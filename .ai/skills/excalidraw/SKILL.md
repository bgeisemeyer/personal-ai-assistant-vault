# Excalidraw Skill

## Activation
- Diagram creation
- Visual refinement
- Canvas-based explanation or planning

## Working Principles
1. Prefer clear structure over decorative complexity.
2. Keep labels readable and specific.
3. Review layout after creating or editing diagrams.
4. Use diagrams to clarify relationships, flow, and boundaries.

## Output Expectations
- Diagram supports the topic clearly.
- Layout remains understandable without verbal explanation.
