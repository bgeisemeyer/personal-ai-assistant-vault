# PDF Processing Skill

## Activation
- Reading external PDFs
- Extracting relevant content into notes
- Documenting external file processing

## Before Processing
1. Read the local `Index.md`.
2. Verify that PDF processing is allowed.
3. Check whether sensitive data may be involved.
4. Ask in unclear or sensitive cases.

## Rules
- Read directly from the source when possible.
- Do not store unnecessary copies in the vault.
- Extract only relevant content.
- Document the processing in the local log.
