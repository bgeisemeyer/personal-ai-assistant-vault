# Vault Conventions

## Purpose
- Keep the framework consistent and easy to continue across AI tools.

## Conventions
- Internal links use `[[...]]`.
- New files start with uppercase names unless the folder is intentionally hidden, such as `.ai/`.
- Context, status, decisions, and sources should be easy to find.
- Prefer small, local structures over broad implicit assumptions.

## Project Principle
- Human-facing knowledge lives in normal vault folders.
- AI-facing framework mechanics can live in hidden folders such as `.ai/`.
