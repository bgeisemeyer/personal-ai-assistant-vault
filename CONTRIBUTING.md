# Contributing

## Principles
- Prefer small, traceable changes.
- Keep framework logic tool-agnostic unless a tool-specific integration is explicitly scoped.
- Preserve file-based context handoff between different AI tools.

## Expected Contribution Areas
- Governance improvements
- New generic skills
- Better templates
- Safer onboarding
- Documentation quality

## Not in Scope by Default
- Personal data
- Secrets
- Tool-specific local runtime files
- Narrowly personal workflows without clear general value
